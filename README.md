# Simple Wordpress Theme
> This is a simple WordPress theme based on a single base template.

The theme is still being developed.

The idea is to create a theme using minimal files.

### What's included?
- Bootstrap 4 CDN
- Popper CDN
- Empty stylesheet

Add plugins as needed.
